# LocalChefBazaar - Home-Cooked Meals Marketplace

A modern platform connecting home chefs with food lovers.

## Live URL
- Client: https://b12a11-imtiaz-local-chef-bazaar.netlify.app
- Server: https://b12a11server.vercel.app/


## Key Features
- JWT-based authentication
- Role-based access control (User, Chef, Admin)
- Real-time order tracking
- Stripe payment integration
- Review and rating system
- Favorite meals
- Dark/Light theme toggle
- Responsive design

## Tech Stack
- Frontend: React, Tailwind CSS, Framer Motion
- Backend: Node.js, Express, MongoDB
- Authentication: Firebase, JWT
- Payment: Stripe
- Others: React Hook Form, Recharts, Axios
